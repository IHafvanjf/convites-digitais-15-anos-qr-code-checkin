# Convites Digitais — 15 Anos com QR Code e Check-in

**Última revisão:** 07-2025

Sistema moderno para gerenciamento de convites digitais de aniversário (15 anos), com **QR Code exclusivo por convidado**, **check-in por câmera** (desktop ou smartphone) e **painel administrativo** para controle de presença e acompanhantes. O fluxo cobre do envio do convite à validação na entrada, com atualização de status em tempo real.

---

## Observação
Projeto desenvolvido para fins **pedagógicos** e portfólio

---

## Destaques
- Convite digital com **QR Code individual**
- **Leitura e validação** por câmera (desktop/mobile) em tempo real
- **Painel seguro** para gerenciamento de convidados e acompanhantes
- **Status do convite**: pendente ou utilizado
- Layout **responsivo** e interface simples

---

## Principais Funcionalidades
- **Geração automática** de QR Codes com os dados do convidado
- **Check-in** com feedback imediato (válido, já utilizado, inválido)
- **Cadastro/edição** de convidados e registro de **acompanhantes**
- **Busca e filtros** por nome, convite e status
- **Relatório rápido** de presença (totais pendentes/utilizados)
- **Envio digital** do convite e reenvio quando necessário

---

## Tecnologias
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP 8+
- **Banco de Dados:** MySQL
- **QR Code:** geração e leitura via biblioteca JS
- **Segurança:** sessões e validação de entrada
